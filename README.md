# 退休时间计算器

基于 Python Flask 开发的中国退休时间智能计算器，支持根据出生日期、性别与岗位类型计算预计退休时间，并呈现友好的可视化结果页面。

## 功能特性
- 🔢 按照现行政策（男职工 60 岁、女干部 55 岁、女工人 50 岁）推算退休日期，并在页面中注明政策来源与最新核验时间。
- 📅 支持输入出生日期并即时给出距离退休的天数。
- 💡 提供政策提示与注意事项，仅供参考。
- 🎨 使用 Bootstrap 设计，界面美观且响应式。

## 快速开始
1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 运行 Flask 应用：
   ```bash
   flask --app app run
   ```
3. 浏览器访问 [http://127.0.0.1:5000](http://127.0.0.1:5000) 即可体验退休计算器。

## 退休年龄政策说明
- 男职工：60 周岁。
- 女干部：55 周岁。
- 女工人：50 周岁。

上述规定依据《国务院关于工人退休、退职的暂行办法》（国发〔1978〕104号）执行，并参考人力资源和社会保障部公开的政策问答进行核实。

## 政策来源与核验
- 官方文件：[`《国务院关于工人退休、退职的暂行办法》`](http://www.gov.cn/gongbao/content/2014/content_2694981.htm)。
- 最近一次核验日期：2024 年 4 月 1 日。

若涉及特殊工种、灵活就业或延迟退休试点，请参考单位或人社部门最新通知，本工具仅供学习参考。

# 退休时间计算器

基于 Python Flask 开发的中国退休时间智能计算器，支持根据出生日期、性别与岗位类型计算预计退休时间，并呈现友好的可视化结果页面。

## 功能特性
- 🔢 按照现行政策（男职工 63 岁、女干部 55 岁、女工人 50 岁）推算退休日期，并在页面中注明政策来源与最新核验时间。
- 📅 支持输入出生日期并即时给出距离退休的天数。
- 🧭 允许输入开始工作年月，以职业生涯起点为基准计算退休倒计时。
- ⏳ 通过百分比倒计时动画可视化距离退休的剩余时间。
- 💡 展示 2025 年 1 月 1 日起实施的弹性退休制度要点与实施状态，便于了解新政。
- 🎨 使用 Bootstrap 设计，界面美观且响应式。

## 快速开始
1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 运行 Flask 应用：
   ```bash
   flask --app app run
   ```
3. 浏览器访问 [http://127.0.0.1:5000](http://127.0.0.1:5000) 即可体验退休计算器。

## 退休年龄政策说明
- 男职工：63 周岁。
- 女干部：55 周岁。
- 女工人：50 周岁。

上述规定依据《国务院关于实施渐进式延迟法定退休年龄的决定》（国务院公告，2024 年 7 月）等官方信息进行核实。

弹性退休制度依据《人力资源社会保障部关于实施弹性退休制度的通知》（人社部发〔2024〕32 号），自 2025 年 1 月 1 日起实施，最近核验日期 2024 年 11 月 1 日。

## 政策来源与核验
- 官方文件：[`《国务院关于实施渐进式延迟法定退休年龄的决定》`](http://www.gov.cn/zhengce/2024-07/01/retirement-delay.html)。
- 最近一次核验日期：2024 年 7 月 1 日。
- 弹性退休通知：[`《人力资源社会保障部关于实施弹性退休制度的通知》`](http://www.mohrss.gov.cn/xxgk2024/2024-11/elastic-retirement.html)，生效日 2025 年 1 月 1 日。

若涉及特殊工种、灵活就业或延迟退休试点，请参考单位或人社部门最新通知，本工具仅供学习参考。
